# odin_project

## concantenation 

### With the plus operator:
"Welcome " + "to " + "Odin!"    #=> "Welcome to Odin!"

### With the shovel operator:
"Welcome " << "to " << "Odin!"  #=> "Welcome to Odin!"

### With the concat method:
"Welcome ".concat("to ").concat("Odin!")  #=> "Welcome to Odin!"

## Substring

"hello"[0]      #=> "h"

"hello"[0..1]   #=> "he"

"hello"[0, 4]   #=> "hell"

"hello"[-1]     #=> "o"

## Capitalize

"hello".capitalize #=> "Hello"
